    • Copperhead Virtual Cluster

    This datacenter based virtual cluster provides a Redhat Linux based high performance computing environment that includes multiple blade servers and systems of various capabilities and serving a variety of research communities. Copperhead (including 59 nodes/944 computing cores) is a general use cluster that is available for use in any faculty sponsored research project.

    Local Physical Cluster

    Our physical cluster has a 192-core big data cluster with a 87TB Hadoop Distributed File System (HDFS) available for use by faculty and graduate student researchers. It consists of 18 nodes (2 Masters, 16 Slaves), each of which has Intel Xeon(R) CPU E5-2630v4@2.20GHz x 20 and 64GB DDR3 RAM, running Ubuntu 16.04 LTS operating system with kernel version 4.0, Scala 2.10.0, and Hadoop YARN 2.8.0 for cluster management.
